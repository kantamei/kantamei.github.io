安装nginx

缺少 PCRE library 和zlib 编译OK会有警告无视

编译nginx OK!
启动错误

解决方法：

ln -s /usr/local/lib/libpcre.so.1 /lib64

启动 ./nginx 输入你的ip 不行 80端口没开 iptables

重新输入OK

安装mysql

yum install -y mysql-community-server

service mysqld start

/usr/bin/mysql_secure_installation 密码不对是因为5.7初始密码不为空两种解决方法
第一种 ， 第二种 OK！

启动后UID NGINX？
创建nginx用户及group useradd nginx

安装phpmyadmin

403错误 那是因为nginx没有添加index.php
填上去继续又是 phpMyAdmin - Error
引起这个错误的原因是 php 没有权限把 session 写到指定的目录里
卧槽们没有？！创建一个 mkdir session

chown vagrant /var/lib/php-fpm/session

ok！

更改端口 8080

/sbin/iptables -I INPUT -p tcp --dport 8080 -j ACCEPT
